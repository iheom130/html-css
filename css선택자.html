class 속성지정

li a href="2.html" class="saw">2/a li

이런식으로 적는다.

그 후에 스타일에가서
.saw{
color:grey;
}
이런식으로 쓰면 됨

코딩은 순서대로 이루어지므로
하나를 빨강으로 그다음 전체 회색하면 당연히 회색이됨
따라서 id="active"로 속성값을 주고
#active 이런식으로 스타일에서 코딩한다면 id선택자가 이기기 때문에 가능함

태그선택자가 제일 약하고 그다음 강한게 클래스, 제일 강한건 id순서임

그냥이 태그 a

.붙인게 클래스.saw
class="saw"

#붙인게 id #active
